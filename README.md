**Nome do Programa:** `CalculadoraAreaCirculo`

**Descrição:**  
Este programa calcula a área de um círculo com base no valor do raio fornecido pelo usuário. Utilizando o valor de π (pi) como 3.14159, o programa realiza o cálculo da área através da fórmula `A = π * raio²` e exibe o resultado com precisão de quatro casas decimais. Ideal para aplicações simples que envolvem cálculos geométricos básicos.
